# PatrickOS

PatrickOS será una distribución personalizada basada en Debian/Ubuntu, optimizada para una ASUS TUF con NVIDIA, enfocada en productividad médica, IA local, automatización y el agente Watson.

## Objetivos iniciales

- Crear una estación de trabajo Linux reproducible.
- Integrar Watson como launcher/agente local.
- Optimizar para NVIDIA, IA local, consulta médica, desarrollo y creación de contenido.
- Mantener el sistema ligero, estable y seguro.

## Fases

1. Entorno de desarrollo en WSL2.
2. Watson mínimo funcional.
3. Scripts de automatización.
4. Configuraciones del sistema.
5. ISO personalizada.
